MTG tools is a website created in 2021 by Force4760 (Gonçalo Teixeira) as a way to test the Svelte Javascript framework.

MTG tools provides to it's users a collection of tools to help on Magic: the Gathering games. This tools include a life counter, a mana tracker, a dice roller, card search and more.

Cards are searched for using the Scryfall API.
